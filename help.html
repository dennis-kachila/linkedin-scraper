LinkedIn Job Scraper - Help Guide

Getting Started

The LinkedIn Job Scraper extension helps you extract and analyze job postings from LinkedIn's job search results. Here's how to use it:

  1. Navigate to LinkedIn Jobs
  2. Perform a job search with your desired criteria (job title, location, etc.)
  3. Click on the extension icon in your browser toolbar
  4. Click the "Start" button to begin scraping job listings
  5. Wait for the scraping to complete (you'll see the progress in the popup)
  6. Click the "Search" button to view and analyze your scraped jobs

Features

Job Scraping

The extension automatically extracts:

  • Job title
  • Company name
  • Location
  • Posted date
  • Number of applicants
  • Full job description
  • Direct link to the job posting

Job Search & Analysis

The Search page provides powerful tools to work with your scraped data:

  • Full-text search across all job details
  • Dedicated description search for finding specific skills or requirements
  • Filtering by keywords in job descriptions
  • Export to CSV or Excel formats
  • Sort by any column

Tips & Troubleshooting

Tip: For the best results, scroll through some job listings on LinkedIn before starting the scraper. This helps LinkedIn load more jobs into the page.

Common Issues

Issue Solution
Scraper doesn't start Make sure you're on a LinkedIn jobs search page. The URL should contain "linkedin.com/jobs"
Progress bar stays at 0% Refresh the page and try again. Make sure you have job listings visible on the page.
Some job details are missing LinkedIn occasionally changes their page structure. Please report this issue on our GitHub page.
Storage limit reached Clear some of your older job data using the "Clear Data" button.
Note: Web scraping should be done responsibly. Avoid making too many requests in a short time period, as this could lead to your account being temporarily limited by LinkedIn.

Privacy & Data

All scraped job data is stored locally in your browser and never sent to external servers. You can clear this data at any time using the "Clear Data" button in either the popup or search page.
